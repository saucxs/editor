# editor

Editor是一款简洁的 Markdown 编辑器。自动将md转成微信公众号图文，拥有一键发送到指定平台能力，支持独立平台独立配置，
支持md样式配置等功能。

## 特性
+ md格式编辑器
+ md格式转换为微信公众格式，一键复制粘贴
+ md格式文档下载
+ 支持选择段落字号
+ 支持选择主题颜色
+ 外链引用开关
+ 上传图片，支持GitHub等多种图床
+ 自定义md格式的css样式
+ 一键发布到多个平台「掘金，微信公众号，知乎等」
+ 自定义不同平台公共头部和尾部
+ 支持用户注册和登陆
+ 支持权限管理
等等

PS： 本项目基于 [wechat-format](https://github.com/lyricat/wechat-format) 进行二次开发，感谢 lyricat 的创意和贡献！
